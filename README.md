# 🔐 Crypto Toolkit

My first cryptography project built with Python.

## Features

- AES Text Encryption
- AES Text Decryption
- RSA Text Encryption
- RSA Text Decryption
- Interactive Command Line Interface (CLI)

## Technologies

- Python 3
- PyCryptodome

## What I Learned

- Symmetric Encryption (AES)
- Asymmetric Encryption (RSA)
- Key Management
- Python Functions
- Command Line Applications

## How to Run

```bash
pip install pycryptodome
python main.py
```

Run the commands from PowerShell or Command Prompt.
